# ShapeCalculationLibrary
Библиотека для расчета площади фигуры.

Для расчета площади фигуры:
1) Необходимо подключить библиотеку в свой проект.
2) Создать инстанс фигуры (на данный момент Circle, Triangle). С передачей необходимых параметров в конструктор.
3) Вызвать метод GetAreaShape из класса ShapeManager. Куда в качестве параметра передать созданную фигуру.
